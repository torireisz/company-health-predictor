# company-health-predictor
Machine learning tool for predicting company financial health and identifying PE investment opportunities

## Company Financial Health & PE Investment Analysis

A machine learning tool that analyzes company financials, predicts financial health, and identifies attractive private equity investment targets using SEC filing data and comparable company analysis.

## Project Overview

This project combines predictive modeling with PE-style financial analysis to:
- Predict company financial health and growth potential
- Calculate key PE valuation metrics (EV/EBITDA, growth rates, margins)
- Provide interactive peer comparisons and industry benchmarking
- Rank companies by PE attractiveness
